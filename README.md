# Aulas JavaScript - Generation Brasil 📚💻

Este repositório contém as atividades e exemplos desenvolvidos durante as aulas de **JavaScript** no **bootcamp de Desenvolvimento FullStack JavaScript da Generation Brasil**.

Durante as aulas, o instrutor apresentava os conceitos e os alunos acompanhavam desenvolvendo os códigos junto com ele.

O objetivo deste repositório é registrar os exercícios realizados durante o processo de aprendizado.

---

# 📚 Conteúdos estudados

Neste repositório estão exemplos e exercícios relacionados a:

- Introdução ao JavaScript
- Variáveis
- Operadores
- Estruturas Condicionais
- Estruturas de Repetição
- Introdução ao Git

# ⚙️ Pré-requisitos

Antes de executar os arquivos, você precisa ter instalado em sua máquina:

- Node.js
- Git

### Instalar Node.js

Baixe pelo site oficial:

https://nodejs.org

Após instalar, verifique no terminal:

```bash
node -v
npm -v
```

## 📥 Como clonar o repositório

No terminal, execute o comando abaixo para clonar o projeto para sua máquina:

```bash
git clone https://github.com/BiiaBraga/Aulas-JavaScript-Generation.git
cd Aulas-JavaScript-Generation
```

## 📦 Instalação das dependências

Após clonar o repositório, é necessário instalar as dependências do projeto.
A pasta `node_modules` não está incluída no repositório pois está listada no `.gitignore`.
Para instalar todas as dependências listadas no arquivo `package.json`, execute no terminal:

```bash
npm install
```

## ▶️ Como executar os arquivos

Este repositório contém arquivos **JavaScript** que podem ser executados utilizando o **Node.js**.

No terminal, utilize o seguinte comando:

```bash
node nomeDoArquivo.js
```
